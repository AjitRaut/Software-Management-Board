Software Management Board Guide

Introduction

Welcome to the Software Management Board. This guide will help you quickly understand and navigate the board.

  • Search Issues: Prioritizes "Progress" tasks.
  • Create Task: Adds a new task and removes dummy data.
  • Profile: View and edit your profile.
  • Logout: Log out of the board.

Task Interaction

Click any task box to scroll down to its detailed info and editing tools.

Summary: Click a task box to view and edit details.

Search Functionality

Use the search input to filter tasks from all categories. The "Search Issues" function prioritizes tasks in the "Progress" category.

Summary: Search tasks across categories; prioritize "Progress" tasks with "Search Issues".

Slider Menu

The slider menu on the left includes:

  • Search Issues: Prioritizes "Progress" tasks.
  • Create Task: Adds a new task and removes dummy data.
  • Profile: View and edit your profile.
  • Logout: Log out of the board.
Summary: Use the slider menu for key functionalities: search, create tasks, profile, and logout.

Profile Section

Add your profile to view credentials. Use "Forgot Password" to reset your password with the correct email.

Example: Click "Forgot Password", enter your email, and reset your password.
Summary: Manage your profile and reset passwords as needed.
